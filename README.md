# 1. 우리가 발견한 문제

최저임금은 근로자의 생활 수준을 보장하기 위해 국가가 정한 최소한의 임금이다. 그러나 최저임금이 크게 인상될 때마다 사회에서는 항상 논쟁이 발생한다.

어떤 사람들은

> "최저임금이 오르면 노동자의 삶이 좋아진다."

고 말한다.

반면 다른 사람들은

> "기업의 인건비 부담이 증가하여 일자리가 줄어들 수 있다."

고 주장한다.

뉴스나 인터넷에서는 다양한 의견이 존재하지만, 실제로 어떤 주장이 더 타당한지는 쉽게 알 수 없었다.

그래서 우리 팀은 단순한 의견이나 느낌이 아니라 실제 데이터를 통해 확인해 보기로 했다.

우리가 정의한 연구 문제는 다음과 같다.

> **"최저임금이 인상될수록 고용률은 감소하는가?"**

그리고 이를 더 구체적으로 확인하기 위해 다음 질문들을 설정하였다.

* 최저임금은 지난 25년 동안 얼마나 증가했는가?
* 고용률은 같은 기간 동안 어떤 변화를 보였는가?
* 두 지표 사이에는 통계적으로 어떤 관계가 존재하는가?
* 최저임금이 크게 인상된 시기에는 실제로 고용이 감소했는가?

# 3. 분석 방법

우리 팀은 단순히 그래프만 비교하는 것이 아니라 여러 방법을 통해 결과를 검증하였다.

### ① 최저임금 변화 분석

연도별 최저임금 추이를 그래프로 시각화하였다.

* 최저임금이 얼마나 증가했는지 확인
* 급격한 인상 시기 파악

---

### ② 고용률 변화 분석

연도별 고용률 추이를 그래프로 나타냈다.

* 장기적인 증가 또는 감소 추세 확인
* 특정 시기의 변화 관찰

---

### ③ 상관관계 분석

최저임금과 고용률 사이의 상관계수를 계산하였다.

이를 통해

> "최저임금이 증가할 때 고용률은 함께 증가하는지, 감소하는지"

를 수치로 확인하였다.

---

### ④ 최저임금 급등 시기 비교

2018년과 2019년은 최저임금이 큰 폭으로 인상된 시기이다.

이 시기의 전후 데이터를 비교하여 실제로 고용률 변화가 있었는지 분석하였다.

# 5. 우리가 내린 결론

분석 결과 최저임금은 2000년 1,600원에서 2025년 10,030원으로 약 6배 이상 증가하였다.

반면 고용률은 같은 기간 동안 58.5%에서 62.9%로 증가하였다.

상관관계 분석 결과 최저임금과 고용률은 양의 상관관계를 보였으며, 최저임금이 상승했다고 해서 고용률이 감소하는 현상은 확인되지 않았다.

또한 최저임금이 크게 인상된 2018년 이후에도 고용률의 급격한 감소는 나타나지 않았다.

따라서 이번 분석 결과만으로는

> "최저임금 인상이 고용 감소를 초래한다"

라고 결론 내리기 어려웠다.

다만 고용률은 경제성장률, 물가상승률, 산업구조 변화, 정부 정책 등 다양한 요인의 영향을 받기 때문에 최저임금만으로 설명할 수 없다는 점도 확인하였다.

# 6. AI를 비판적으로 활용한 과정

이번 프로젝트에서는 ChatGPT를 활용하여 데이터 분석 아이디어와 스트림릿 웹앱 코드를 작성하였다.

하지만 AI의 답변을 그대로 사용하지 않고 여러 차례 검증 과정을 거쳤다.

* AI가 제안한 데이터 분석 방법이 실제 경제학 연구에서도 사용되는지 확인하였다.
* AI가 계산한 결과를 원본 데이터와 비교하여 검증하였다.
* 상관관계가 높다고 해서 인과관계가 성립하는 것은 아니라는 점을 확인하였다.
* 최저임금이 고용에 영향을 미친다는 AI의 설명을 그대로 수용하지 않고 실제 데이터를 통해 검토하였다.

특히 AI가 제시한 결론보다 실제 데이터 분석 결과를 우선적으로 해석하려고 노력하였다.

> 💡 AI는 답을 주는 존재가 아니라, 문제 해결을 돕는 도구라는 점을 배울 수 있었다.

# 9. 이 분석의 한계

이번 프로젝트는 실제 데이터를 활용하였지만 몇 가지 한계가 존재한다.

* 고용률 외에 경제성장률, 물가상승률 등의 변수는 고려하지 못하였다.
* 상관관계 분석만으로 인과관계를 증명할 수 없다.
* 전체 고용률만 분석하였으며 산업별·연령별 영향은 충분히 분석하지 못하였다.
* 최저임금의 효과가 즉시 나타나지 않을 수도 있다는 점을 고려하지 못하였다.

한계를 인정하고 함께 제시하는 것은 올바른 데이터 분석 과정의 중요한 부분이라고 생각한다.
